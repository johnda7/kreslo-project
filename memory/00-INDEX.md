# Память проекта kreslo-project — карта

> Читай ПЕРВЫМ при старте сессии. Здесь — как устроена память проекта и куда что писать.

## Что где

| Файл | Назначение |
|------|-----------|
| `../CLAUDE.md` | Технический контекст: форма, нарезка, материалы, физика, посадка, метод скана |
| `memory/00-INDEX.md` | Эта карта |
| `memory/business.md` | Зачем проект: деньги, связь с Пхукетом, монетизация, Instagram, позиционирование |
| `memory/log.md` | Журнал работы: что делали, решения, извлечённое из скинутых чатов |

## Как пополнять память
- Джон скидывает старые чаты сюда → извлекаю важное → пишу в `log.md` (хронология решений) и в `../CLAUDE.md` (если это стабильный технический факт: размер, материал, метод).
- Гипотезы помечать словом «гипотеза». Замеры/цифры — только подтверждённые.
