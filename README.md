# 🪑 Кресло-чаша · DIY fiberglass lounge chair

Самодельное дизайнерское кресло-чаша из стеклопластика (по мотивам UFO Mavimatt).
Проект: проектирование формы из LiDAR-скана → нарезка пенопластового болвана → стеклопластик.

## Быстрый старт

1. Открой `CLAUDE.md` — там весь контекст проекта (форма, размеры, нарезка, материалы, технология).
2. Открой `site/kreslo-site.html` в браузере — интерактивный конфигуратор:
   - верхний 3D — целевая чаша Ø110 (качается по кругу)
   - нижний 3D — реальное вырезанное кресло (облако точек LiDAR, вращается)
   - ползунок высоты, таблица слоёв, план нарезки, раскладка листов
3. `images/` — все чертежи (раскладка кроя, схема слоёв, посадка, сравнение форм).
4. `data/` — профили, облака точек, исходные LiDAR-сканы (.usdz).

## Разбор LiDAR-скана

```bash
pip install usd-core numpy matplotlib --break-system-packages
python tools/parse_scan.py data/scan-kreslo.usdz
```
