## ЧТО ЭТО

Джон (Пхукет, Таиланд) делает СВОИМИ РУКАМИ дизайнерское кресло-чашу методом стеклопластика по мотивам кресла **UFO (Mavimatt)** и формы реальной керамической чашки. Первый прототип уже вырезан и склеен из пенопласта, залит гипсом, сейчас на стадии шкурения.

## КОНЦЕПЦИЯ ФОРМЫ

Глубокая чаша-полусфера. Два варианта в работе:
- **Вариант А (цель, по чашке-референсу):** Ø110 см, высота 49 см, дно 39 см. Под него рассчитана вся нарезка (11 слоёв).
- **Вариант Б (что реально вышло, по LiDAR-скану готового кресла):** яйцевидное/вытянутое — длина с балансиром ~161 см, ширина вбок ~79 см, высота ~101 см. Основная чаша ~Ø130, сзади хвост-балансир.

**Цель сейчас:** сделать СРЕДНЕЕ между А и Б + решить главную проблему — поднять посадку (низко, тяжело вставать).

## ФИЗИКА / КОНСТРУКЦИЯ

- Чаша наклонена на балансир под углом **54° (обод к полу)**, ось чаши 36°. Угол рассчитан из формы (опора на бок + край донышка), НЕ выдуман.
- **Качание по кругу** (не вперёд-назад): постоянный наклон, точка опоры обходит по окружности. Как неваляшка.
- Балансир = заднее донышко Ø36 / h14 см — отдельная деталь, ещё НЕ делали.
- Реальное кресло Джон поставил на ножку (не качается, вращается) — отсюда задача с высотой посадки.

## ПОСАДКА (расчёт для Ø110, человек 187 см)

- Таз/сиденье: 22 см от пола
- Спинка/борт: 86 см (по плечи)
- Голова: 119 см — НАД чашей (не кокон!)
- Ноги через передний край (21 см) свисают на пол
- ГОЛОВА И ПЛЕЧИ НАД ЧАШЕЙ, внутри только таз+спина до лопаток.
- Хитрость UFO для лёгкого вставания: сиденье НЕглубокое и приподнятое (таз высоко, не проваливается).

## КРИТИЧНО ПО ТЕХНОЛОГИИ

- **Полиэфирная смола РАСТВОРЯЕТ пенопласт EPS** (стирол). Барьер ОБЯЗАТЕЛЕН: гипс по пене → сохнет → потом стеклопластик. (Джон уже залил гипсом.)
- MEKP ~1-2% от веса смолы. В жару Пхукета реакция быстрая — мешать малыми порциями, время жизни замеса 10-15 мин.
- Смолу лить в одноразовую посуду из PP/PE (НЕ в пенопластовые стаканы — растворятся). Мешалки деревянные.
- Под пол — плотная полиэтиленовая плёнка/брезент от капель.
- Работать на улице/балконе, респиратор обязателен (пары стирола).

## КАК РАБОТАТЬ СО СКАНОМ (метод отработан)

USDZ читается через `usd-core` (pip install usd-core --break-system-packages).
Чаша сканируется под наклоном → истинную ось симметрии искать через **PCA** (ось вращения = собственный вектор с наименьшим собственным значением для тела вращения). Профиль строить вдоль этой оси: percentile(r, 90) по уровням высоты = внешний контур.

## СЛЕДУЮЩИЕ ШАГИ

1. Джон шкурит гипс на готовом кресле.
2. Замерить рулеткой высоту ножки готового кресла (по скану смазано).
3. Решить подъём посадки — ТРИ рычага: (1) ножка-подставка поднимает всю чашу (как UFO, главный подъём); (2) толстая плотная подушка на дно 15-20 см (поднимает точку посадки, не проваливаешься); (3) не делать чашу глубокой (сиденье ближе к краю). Цель — сиденье ~38-42 см от пола, неглубокое, таз выше колен → легко вставать.
4. Спроектировать балансир/ножку отдельно.
5. Барьер (готов) → гелькоут → стекломат+смола 6-8 мм снаружи → шлифовка → покраска.
6. Возможно — следующее кресло среднего размера между Ø110 и реальным.
